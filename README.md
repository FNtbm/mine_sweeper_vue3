## 扫雷小游戏（MineSweeper） ....正在开发中

借用 antfu 大佬的[VitesseLite](https://github.com/antfu/vitesse-lite)开发
- netlify部署 https://minesweeper-lite.netlify.app

- :car: 借用该环境

  ```command
  degit antfu/vitesse-lite
  ```

- :runner: 启动该项目

  ```command
  pnmp install

  pnmp run dev
  ```

  **or**

- :bicyclist: 使用 [ni](https://github.com/antfu/ni)

  ```js
    npm i @antfu/ni -g

    ni == npm install

    nr dev == npm run dev
  ```
